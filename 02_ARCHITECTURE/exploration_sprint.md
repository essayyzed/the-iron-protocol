## Week 1-2: Machine Learning & AI

### What You'll Explore

- Supervised vs unsupervised learning
- Neural networks basics
- Popular ML libraries (scikit-learn, TensorFlow)
- Real-world ML applications

### Mini Project: Image Classifier

**Goal:** Build a model that classifies images (e.g., cats vs dogs)

**Tech Stack:**
- Python
- TensorFlow/Keras or PyTorch
- Jupyter Notebook

**Steps:**
1. Find a dataset (Kaggle, ImageNet)
2. Preprocess images
3. Build a simple CNN
4. Train and evaluate
5. Test with your own images

**Resources:**
- [Fast.ai Practical Deep Learning](https://course.fast.ai/)
- [Google's ML Crash Course](https://developers.google.com/machine-learning/crash-course)

---

## Week 3-4: Mobile Development

### What You'll Explore

- iOS vs Android vs Cross-platform
- Mobile UI/UX principles
- React Native or Flutter
- App deployment to stores

### Mini Project: Todo App or Weather App

**Goal:** Build and deploy a mobile app

**Tech Stack:**
- React Native (JavaScript) or Flutter (Dart)
- Expo (for React Native)

**Features:**
- Clean UI
- Local storage (AsyncStorage)
- API integration
- Works on both iOS and Android

**Resources:**
- [React Native Tutorial](https://reactnative.dev/docs/tutorial)
- [Flutter Cookbook](https://docs.flutter.dev/cookbook)

---

## Week 5-6: Game Development

### What You'll Explore

- Game engines (Unity, Unreal, Godot)
- Game physics and collision detection
- 2D vs 3D game development
- Game design patterns

### Mini Project: 2D Platformer or Puzzle Game

**Goal:** Build a playable game

**Tech Stack:**
- Unity (C#) or Godot (GDScript)
- Free assets from itch.io

**Features:**
- Player movement
- Obstacles/enemies
- Score system
- At least 3 levels

**Resources:**
- [Brackeys Unity Tutorials](https://www.youtube.com/@Brackeys)
- [Godot Tutorials](https://docs.godotengine.org/en/stable/community/tutorials.html)

---

## Week 7-8: Cloud & DevOps

### What You'll Explore

- Cloud providers (AWS, Azure, GCP)
- Infrastructure as Code
- CI/CD pipelines
- Docker and Kubernetes basics

### Mini Project: Automated Deployment Pipeline

**Goal:** Deploy an app with full CI/CD

**Tech Stack:**
- GitHub Actions or GitLab CI
- Docker
- AWS/Heroku/Railway

**Steps:**
1. Containerize an app (from previous semester)
2. Write Dockerfile
3. Set up GitHub Actions
4. Auto-deploy on push to main
5. Add tests to pipeline

**Resources:**
- [Docker Tutorial](https://docs.docker.com/get-started/)
- [GitHub Actions Tutorial](https://docs.github.com/en/actions)

---

## Week 9-10: Cybersecurity

### What You'll Explore

- Common vulnerabilities (OWASP Top 10)
- Penetration testing basics
- Cryptography fundamentals
- Security tools (Burp Suite, Nmap, Wireshark)

### Mini Project: Vulnerable App + Security Audit

**Goal:** Build an intentionally vulnerable app, then secure it

**Tech Stack:**
- Node.js/Python
- OWASP Juice Shop (for practice)

**Tasks:**
1. Identify SQL injection vulnerabilities
2. Fix them with parameterized queries
3. Add authentication with hashing
4. Implement HTTPS
5. Write security report

**Resources:**
- [OWASP Top 10](https://owasp.org/www-project-top-ten/)
- [TryHackMe](https://tryhackme.com/)
- [HackTheBox](https://www.hackthebox.com/)
